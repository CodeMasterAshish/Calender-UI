📅 Calendar UI (HTML & CSS)
A simple and responsive Calendar User Interface built using only HTML5 and CSS3. This project recreates a calendar layout similar to a date picker component without using JavaScript.
🚀 Features


Date input field with calendar icon


Monthly calendar layout


Clean and minimal design


Built using CSS Grid and Flexbox


Hover effect on dates


Beginner-friendly project


No JavaScript required


🛠️ Technologies Used


HTML5


CSS3
